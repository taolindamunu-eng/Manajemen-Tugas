# Manajemen Tugas

## Deskripsi
**Manajemen Tugas** adalah aplikasi berbasis web yang dibangun menggunakan framework Laravel untuk membantu pengguna dalam mengelola tugas seperti menambahkan, mengedit, dan menghapus tugas.

---

## Fitur Utama
- Tambah tugas
- Edit tugas
- Hapus tugas
- Tampilan sederhana dan mudah digunakan

---

## Teknologi
- Laravel (PHP Framework)
- MySQL / Database
- HTML, CSS, JavaScript

---

## Cara Menjalankan

```bash
composer install
cp .env.example .env
php artisan key:generate
php artisan serve
